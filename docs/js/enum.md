# enum

Enum objects with helper functions

## Example
    module.exports = Enum.extend({
      jpg: 'jpg',
      png: 'png',
      gif: 'gif',
      jpe: 'jpg',
      jpeg: 'jpg'
    });

## Exposed functions
* keys
  * get all the enum keys
* values
  * get all the enum values
* contains
  * check if an value exists
